# 📊 Sales Performance Analysis (Excel Dashboard)

## 📌 Project Overview
This project analyzes sales data using Excel to create an interactive dashboard for business insights and performance tracking.

## 🎯 Objectives
- Analyze sales performance across regions
- Identify top-selling products
- Track quarterly growth
- Automate reporting process

## 📊 Key Features
- Interactive Dashboard with Slicers
- Region-wise Analysis
- Product-wise Performance
- Quarterly Sales Trends

## 🛠 Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Power Query
- Charts & Graphs

## 📈 Key Insights
- 22% sales growth in Q3
- Identified top-performing regions
- Highlighted underperforming products

## 📁 Dataset
- 15,000+ rows of sales data

## 🚀 How to Use
1. Open Excel file
2. Use slicers to filter data
3. Analyze charts and KPIs

## 📌 Conclusion
This project improves reporting efficiency and provides clear insights for business decision-making.
